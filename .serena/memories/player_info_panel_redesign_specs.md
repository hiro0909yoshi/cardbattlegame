# Player Info Panel Redesign - UI Specs

## 1. パネル配置・サイズ
- 配置: 縦積み（左上から順に）、間隔10px
- サイズ: 160×105px
- フォントサイズ: 15px

## 2. 表示項目
- プレイヤー名
- 魔力（現在値）
- 総魔力（総資産の既存計算をそのまま使用、名前だけ「総資産」→「総魔力」に変更）

## 3. パネル色
- GameConstants.PLAYER_COLORS配列を使用（Player 1=黄色、Player 2=青など）

## 4. ステータスダイアログ内容

### 土地情報
- 属性ごとに土地数を表示（例：「火×3個」「水×2個」）

### 保有クリーチャー
- 名前
- 現在HP
- 最大HP
- 現在AP

### 属性連鎖
- 不要（土地属性情報で十分）
