# クリーチャー画像フォルダ

## 概要
このフォルダにはクリーチャーカードの画像を配置します。

## 命名規則
ファイル名はクリーチャーのIDと一致させてください：

```
1.png  → アームドパラディン (ID: 1)
2.png  → アモン (ID: 2)
3.png  → アングリーモブ (ID: 3)
...
```

## 画像仕様
- **推奨サイズ**: 100x98ピクセル以上（ほぼ正方形）
- **形式**: PNG（透過対応）またはJPG
- **注意**: カード内で自動的にリサイズされます

## 使用方法
1. このフォルダに `{ID}.png` という名前で画像を配置
2. ゲーム実行時、カードは自動的に画像を読み込みます
3. 画像がない場合は、属性に応じた色のプレースホルダーが表示されます

## 例
```
assets/images/creatures/
  ├── 1.png   # 火属性クリーチャー
  ├── 2.png   # 火属性クリーチャー
  ├── 50.png  # 水属性クリーチャー
  └── ...
```
