# スキル・刻印名リネーム トラッキング

## 概要

カルドセプト風の名前を独自の名前に変更する作業の管理ドキュメント。

**方針**:
- 一般的なRPG用語（再生、即死、反射など）→ 変更不要
- カルドセプト固有の名前 → 変更必要
- 呪い → **「刻印」に概念変更**（アルカナ・タロット風の世界観）
- 世界刻印はタロット大アルカナ名を使用

**タロット大アルカナ割り当て**:

| タロット | 割り当て先 |
|---|---|
| 愚者 | 世界刻印#6（召喚条件解除） |
| 女教皇 | 世界刻印#4（刻印守護） |
| 女帝 | 世界刻印#7（全域結界） |
| 皇帝 | 世界刻印#3（土地保護） |
| 恋人 | 世界刻印#2（属性連鎖） |
| 隠者 | 世界刻印#8（同種相殺） |
| 吊人 | 世界刻印#5（能力封じ） |
| 太陽 | 世界刻印#1（コスト上昇） |
| 節制 | 世界刻印#9（侵攻制限） |
| **未使用** | 魔術師, 教皇, 力, 運命の輪, 正義, 死神, 星, 月, 塔, 審判, 世界, 戦車, 悪魔 |

**作業フロー**:
1. 新しい名前を決定 → 「新名」列に記入
2. 全ファイルを検索して置換実施
3. 置換完了したら各チェックボックスを `[x]` に更新
