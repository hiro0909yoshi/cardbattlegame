# クリーチャー移動システム

**バージョン**: 2.0  
**最終更新**: 2025年11月29日  
**ステータス**: 実装完了

---

## 概要

クリーチャーをマップ上で移動させるスペル・秘術のシステム。移動時は呪いが消滅する。

---

## 実装済みスペル・秘術

### スペル

| ID | 名前 | 効果 | 効果タイプ |
|----|------|------|-----------|
| 2002 | アウトレイジ | 対象クリーチャーを隣接敵領地に移動 | `move_to_adjacent_enemy` |
| 2052 | チャリオット | 自クリーチャーを2マス移動；復帰[ブック] | `move_steps` |

### 秘術

| ID | クリーチャー | 効果 | 効果タイプ |
|----|-------------|------|-----------|
| 10 | クリーピングフレイム | G30・1マス移動（防御型） | `move_self` |
| 21 | デスリーチ | G80・HP減少+ダウンした敵を破壊し移動 | `destroy_and_move` |
| 218 | ジャングラバー | G30・敵に移動不可呪い付与 | `creature_curse` |
| 28 | バードメイデン | G50・自クリーチャーに遠隔移動呪い付与 | `creature_curse` |
| 322 | スレイプニール | G60・自クリーチャーを1~2マス移動 | `move_steps` |

## 呪いシステム

### 移動不可呪い（move_disable）

- **付与**: ジャングラバー秘術（spell_id: 9013）
- **効果**: 通常移動・スペル/秘術移動すべて不可
- **チェック箇所**:
  - MovementHelper: 通常移動時
  - SpellCreatureMove: スペル/秘術移動時
  - TargetSelectionHelper: 移動系スペルの対象選択時（`can_move: true`）
  - SpellMysticArts: 移動系秘術の使用可否

### 遠隔移動呪い（remote_move）

- **付与**: バードメイデン秘術（spell_id: 9014）
- **効果**: 領地コマンドで全空き地に移動可能
- **チェック箇所**: MovementHelper._detect_move_type()

### 呪い消滅

移動すると呪いは消滅する。処理箇所：
- LandActionHelper.confirm_move(): 領地コマンド移動時
- SpellCreatureMove._execute_move(): スペル/秘術移動時
- MovementHelper.execute_creature_move(): 汎用移動時

---

## 処理フロー

### 空き地への移動

```
SpellCreatureMove._apply_move_*()
  → 移動先候補取得
  → 移動先選択UI
  → _execute_move()
	→ 呪い消滅
	→ クリーチャー移動
	→ ダウン状態設定
```

### 敵領地への移動（戦闘発生）

```
SpellCreatureMove._apply_move_*()
  → 移動先候補取得
  → 移動先選択UI
  → { trigger_battle: true } を返す

SpellPhaseHandler._apply_single_effect()
  → BattleSystem.execute_3d_battle_with_data(from_tile_index)

BattleSystem._apply_post_battle_effects()
  → 勝利: 移動先に配置、移動元削除
  → 敗北: 移動元のクリーチャー削除
  → 相打ち: 両方削除
```

## JSON定義

### アウトレイジ（2002）

```json
{
  "effect_parsed": {
	"target_type": "creature",
	"target_info": {
	  "owner_filter": "any",
	  "has_adjacent_enemy": true,
	  "can_move": true
	},
	"effects": [{ "effect_type": "move_to_adjacent_enemy" }]
  }
}
```

### チャリオット（2052）

```json
{
  "effect_parsed": {
	"target_type": "creature",
	"target_info": { "owner_filter": "own", "can_move": true },

	"effects": [{ "effect_type": "move_steps", "steps": 2, "exact_steps": true }]
```

## target_info条件一覧

| 条件 | 説明 |
|------|------|
| `can_move` | 移動不可呪いを持たないクリーチャーのみ |
| `has_adjacent_enemy` | 隣接敵領地を持つクリーチャーのみ |
| `hp_reduced` | HPが最大値未満のクリーチャーのみ |
| `is_down` | ダウン状態のクリーチャーのみ |
| `exclude_enemy_creatures` | 敵クリーチャーがいるタイルを除外（防御型用） |

---

## 特記事項

- **チェックポイント・ワープ**: 通過可能だが停止不可
- **復帰[ブック]**: `return_to_deck: true`で使用後デッキに戻る
- **防御型の移動**: 敵クリーチャーがいるタイルには移動不可
- **秘術の2段階選択**: スレイプニールは移動対象→移動先の順で選択
