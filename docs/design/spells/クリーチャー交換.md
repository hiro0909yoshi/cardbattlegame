# クリーチャー交換

## 概要

クリーチャーの位置や配置を交換するスペル・秘術。

## 効果タイプ

| effect_type | 説明 | 対象スペル |
|-------------|------|-----------|
| `swap_with_hand` | 盤面クリーチャーと手札クリーチャーを交換 | エクスチェンジ |
| `swap_board_creatures` | 盤面上の2体のクリーチャーを交換 | リリーフ |
| `swap_creature` | 手札間のクリーチャー交換（奪取） | レムレース秘術 |
