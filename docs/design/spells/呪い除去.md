# 呪い除去システム

## 概要
呪いを除去するスペルと秘術の実装。

## 実装ファイル
- `scripts/spells/spell_purify.gd` - 呪い除去クラス
- `data/spell_mystic.json` - 秘術定義（9024〜9026）

## スペル

| ID | 名前 | コスト | 効果 |
|----|------|--------|------|
| 2073 | ピュアリファイ | MP20 | 全呪いを消し、消した呪いの種類×G50を得る |

### 対象となる呪い
- クリーチャー呪い（全タイル）
- プレイヤー呪い（全プレイヤー）
- 世界呪い

### 魔力獲得の計算
- 同じ呪いタイプが複数あっても1種類としてカウント
- 例: 3体に「戦闘行動不可」→ 1種類 = G50

## 秘術

| ID | 名前 | コスト | 効果 | 使用クリーチャー |
|----|------|--------|------|------------------|
| 9024 | 領地呪い除去 | G30 | 対象領地の呪いを消す | アーキビショップ(403) |
| 9025 | 世界呪い除去 | G50 | 世界呪いを消す | ウリエル(4) |
| 9026 | 全プレイヤー呪い除去 | G30 | 全セプターの呪いを消す | シャラザード(17) |

## effect_type一覧

| effect_type | 説明 |
|-------------|------|
| `purify_all` | 全呪い除去（2073） |
| `remove_creature_curse` | 領地呪い除去（秘術9024） |
| `remove_world_curse` | 世界呪い除去（秘術9025） |
| `remove_all_player_curses` | 全プレイヤー呪い除去（秘術9026） |

## SpellPurifyクラス メソッド

```
purify_all(caster_id) -> Dictionary
  全呪いを除去し、種類×G50を獲得
  戻り値: {removed_types: Array, gold_gained: int}

remove_creature_curse(tile_index) -> Dictionary
  対象領地のクリーチャー呪いを除去

remove_world_curse() -> Dictionary
  世界呪いを除去

remove_all_player_curses() -> int
  全プレイヤーの呪いを除去
  戻り値: 除去した呪いの数
```

## 関連システム
- `spell_curse.gd` - 呪い付与・管理
- `spell_curse_stat.gd` - ステータス変更呪い
- `spell_curse_battle.gd` - 戦闘関連呪い
- `spell_curse_toll.gd` - 通行料関連呪い
